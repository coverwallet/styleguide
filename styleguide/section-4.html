4.2 Different inputs text, password, email

Add any of the following classes to the .form-input base class.

.form-input--long
Long form
.form-input--error
Show error style
.form-input--email
Email icon
.form-input--password
Password icon 
<form class="form">
   <div class="form-group">
       <label for="email">Email</label>
       <input type="text" id="email" class="form-input [modifier class]" placeholder="Email"/>
   </div>
</form>
