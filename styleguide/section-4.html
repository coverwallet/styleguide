4 Standard buttons

Button styles can be applied to anything with the .btn class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

4.1 Base styles & states

:hover
Hover state.
:active
Active state.
.disabled
Disabled button. If you use <button> element, that is acceptable to add disabled attribute instead.
.btn--blue
Blue button
.btn--blue:hover
Blue button (Hover state)
.btn--blue:active
Blue button (Active state)
.btn--blue.disabled
Blue button (Disabled state) 
<button type="submit" class="btn [modifier class]">Button</button>

4.2 Button group

<div class="btn-group">
    <button type="submit" class="btn">Button1</button>
    <button type="submit" class="btn">Button2</button>
</div>
